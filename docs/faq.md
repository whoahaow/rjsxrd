# Часто задаваемые вопросы

## Общие вопросы

### Что такое rjsxrd?
rjsxrd - это проект, который автоматически собирает, фильтрует и обновляет публичные VPN-конфиги для обхода блокировок. Конфиги обновляются каждые 12 часов.

### Как часто обновляются конфиги?
Конфиги обновляются автоматически каждые 12 часов с помощью GitHub Actions.

### Почему файлы разделены?
Большие файлы (особенно 26.txt) разделены на части по 300 конфигов для улучшения производительности на мобильных устройствах и избежания лагов при импорте.

## Технические вопросы

### Какие протоколы поддерживаются?
Поддерживаются наиболее популярные протоколы: V2Ray, VLESS, Trojan, VMess, Shadowsocks, а также специализированные протоколы.

### Что такое "Обход SNI/CIDR белых списков"?
Это специальные конфиги (файлы 26+), которые используют SNI домены из белых списков для обхода ограничений.

### Как работает фильтрация?
Фильтрация происходит по доменам из белых списков (например, домены Avito, Yandex, Mail.ru и др.) для создания конфигов, способных обойти определенные блокировки.

## Проблемы и решения

### Не подключается ни один конфиг
- Попробуйте перезапустить приложение и устройство
- Проверьте настройки брандмауэра и антивируса
- Обновите список конфигов в приложении

### Медленное подключение
- Выберите конфиг с меньшим пингом
- Попробуйте разные протоколы
- Проверьте качество интернет-соединения

### Проблемы с импортом на мобильном
- Импортируйте конфиги по отдельности, особенно файлы 26+ 
- Разделите большой импорт на несколько частей
- Обновите приложение до последней версии

## Безопасность

### На сколько безопасны эти конфиги?
Конфиги являются публичными и берутся из открытых источников. Используйте на свой страх и риск. Рекомендуется использовать для просмотра заблокированного контента, а не для чувствительных действий.

### Нужно ли доверять этим конфигам?
Как и с любыми публичными VPN-конфигами, рекомендуется использовать их с пониманием рисков. Они не обеспечивают полной анонимности.
