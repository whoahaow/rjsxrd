# Часто задаваемые вопросы

## Общие вопросы

### Что такое rjsxrd?
rjsxrd - это проект, который автоматически собирает, фильтрует и обновляет публичные VPN-конфиги для обхода блокировок. Конфиги обновляются каждые 12 часов.

### Как часто обновляются конфиги?
Конфиги обновляются автоматически каждые 12 часов с помощью GitHub Actions.

### Почему файлы разделены?
Все конфиги для обхода SNI/CIDR белых списков теперь доступны в одном файле: all.txt. Если возникают проблемы с производительностью при импорте большого файла all.txt, особенно на мобильных устройствах, файл разделен на части по 300 конфигов (файлы 26.txt, 27.txt и т.д.) для улучшения производительности и избежания лагов при импорте.

## Технические вопросы

### Какие протоколы поддерживаются?
Поддерживаются наиболее популярные протоколы: V2Ray, VLESS, Trojan, VMess, Shadowsocks, Hysteria2, TUIC, а также специализированные протоколы.

### Что такое "Обход SNI/CIDR белых списков"?
Это специальные конфиги, которые используют SNI домены из белых списков для обхода ограничений. Все эти конфиги доступны в общем файле all.txt, а также разбиты на части по 300 конфигов в файлах 26+.

### Какие типы источников конфигов поддерживаются?
Поддерживаются обычные ссылки на конфиги, base64-кодированные подписки (base64-сабы), а также конфиги с фильтрацией по белым доменам для обхода SNI/CIDR блокировок.

### Как обеспечивается безопасность?
Все конфиги с insecure параметрами (allowInsecure=true, insecure=1 и др.) фильтруются из итоговых файлов для предотвращения потенциальных уязвимостей.

### Как работает фильтрация?
Фильтрация происходит по доменам из белых списков (например, домены Avito, Yandex, Mail.ru и др.) для создания конфигов, способных обойти определенные блокировки.

## Проблемы и решения

### Не подключается ни один конфиг
- Попробуйте перезапустить приложение и устройство
- Проверьте настройки брандмауэра и антивируса
- Обновите список конфигов в приложении

### Медленное подключение
- Выберите конфиг с меньшим пингом
- Попробуйте разные протоколы
- Проверьте качество интернет-соединения

### Проблемы с импортом на мобильном
- Сначала попробуйте импортировать файл all.txt (все конфиги для обхода SNI/CIDR в одном файле)
- Если возникают проблемы с производительностью, импортируйте конфиги по отдельности, особенно файлы 26+ (разделенные части)
- Разделите большой импорт на несколько частей
- Обновите приложение до последней версии

## Безопасность

### На сколько безопасны эти конфиги?
Конфиги являются публичными и берутся из открытых источников. Используйте на свой страх и риск. Рекомендуется использовать для просмотра заблокированного контента, а не для чувствительных действий.

### Нужно ли доверять этим конфигам?
Как и с любыми публичными VPN-конфигами, рекомендуется использовать их с пониманием рисков. Они не обеспечивают полной анонимности.
