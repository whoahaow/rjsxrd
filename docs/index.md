## Краткий обзор

rjsxrd - это автоматически обновляемая коллекция публичных VPN-конфигов, предназначенных для обхода блокировок. Проект генерирует и поддерживает актуальные конфигурации, которые можно импортировать в различные VPN-клиенты.

Ключевые особенности:
- Автоматическое обновление каждые 12 часов
- Поддержка различных протоколов
- Обработка base64-кодированных подписок
- Фильтрация конфигов с insecure параметрами
- Разделение больших файлов для лучшей производительности
- Специальные конфиги для обхода SNI/CIDR белых списков

Для начала работы рекомендуется ознакомиться с разделом [Установка и использование](installation.md).
