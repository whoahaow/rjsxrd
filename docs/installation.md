# Установка и использование

## Быстрая настройка

1. Для обхода SNI/CIDR белых списков: используйте файл all.txt (все конфиги в одном файле), или файлы 26+ (разделенные по 300 серверов). Скопируйте нужную ссылку из раздела "Общий список всех вечно актуальных конфигов" в README
2. Импортируйте её в ваш VPN-клиент (v2rayNG, NekoRay, Throne и др.)
3. Выберите сервер с минимальным пингом и подключайтесь

## Использование на мобильных устройствах

### Android
1. Установите приложение v2rayNG
2. Скопируйте ссылку на конфиг в буфер обмена
3. В приложении нажмите "+" → "Импорт из буфера обмена"
4. Для обхода SNI/CIDR белых списков рекомендуется сначала попробовать файл all.txt (все конфиги в одном файле), а при возникновении лагов использовать файлы 26+ (разделенные по 300 серверов)
5. Выберите нужный сервер и подключитесь

### iOS
1. Установите приложение V2Box или другое поддерживаемое
2. Используйте функцию импорта по ссылке
3. Для обхода SNI/CIDR белых списков рекомендуется сначала попробовать файл all.txt (все конфиги в одном файле), а при возникновении лагов использовать файлы 26+ (разделенные по 300 серверов)
4. Выберите сервер с наилучшими характеристиками

## Использование на компьютере

### Windows
1. Установите приложение Throne или v2rayN
2. Импортируйте конфиг по ссылке
3. Запустите подключение

### Linux/MacOS
1. Настройте клиент v2ray
2. Используйте скопированную ссылку для добавления конфига
3. Подключитесь к выбранному серверу
