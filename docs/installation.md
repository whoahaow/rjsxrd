# Установка и использование

## Быстрая настройка

1. Для обычных конфигов: используйте файлы из папки default/ (например, 1.txt, 6.txt, 22.txt и т.д.)
2. Для обхода SNI/CIDR белых списков: используйте файл bypass-all.txt (все безопасные конфиги в одном файле) или файлы bypass-1.txt, bypass-2.txt и т.д. (разделенные по 300 серверов)
3. Для небезопасных конфигов обхода SNI/CIDR: используйте файл bypass-unsecure-all.txt или файлы bypass-unsecure-1.txt и т.д.
4. Для протокол-специфичных конфигов: используйте файлы из папки split-by-protocols/ (например, vless.txt, vmess-secure.txt и т.д.)
5. Импортируйте ссылку в ваш VPN-клиент (v2rayNG, NekoRay, Throne и др.)
6. Выберите сервер с минимальным пингом и подключайтесь

## Использование на мобильных устройствах

### Android
1. Установите приложение v2rayNG
2. Скопируйте ссылку на конфиг в буфер обмена
3. В приложении нажмите "+" → "Импорт из буфера обмена"
4. Для обхода SNI/CIDR белых списков рекомендуется сначала попробовать файл bypass-all.txt (все безопасные конфиги в одном файле), а при возникновении лагов использовать файлы bypass-1.txt, bypass-2.txt и т.д. (разделенные по 300 серверов)
5. Выберите нужный сервер и подключитесь

### iOS
1. Установите приложение V2Box или другое поддерживаемое
2. Используйте функцию импорта по ссылке
3. Для обхода SNI/CIDR белых списков рекомендуется сначала попробовать файл bypass-all.txt (все безопасные конфиги в одном файле), а при возникновении лагов использовать файлы bypass-1.txt и т.д. (разделенные по 300 серверов)
4. Выберите сервер с наилучшими характеристиками

## Использование на компьютере

### Windows
1. Установите приложение Throne или v2rayN
2. Импортируйте конфиг по ссылке
3. Запустите подключение

### Linux/MacOS
1. Настройте клиент v2ray
2. Используйте скопированную ссылку для добавления конфига
3. Подключитесь к выбранному серверу
